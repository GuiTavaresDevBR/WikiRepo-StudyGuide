HTML 5

HTML 5 - é uma linguagem de marcação construída a base de tags. Aqui é o pilar do FRONT-END. Aqui é atravé sde suas TAGS, iremos estruturar a semântica de nossos sites. Tudo aprendido aqui será utilizado futuramente em frameworks mais poderosos como React, etc. HTML5 é comumente uma das primeiras tecnolgoias a serem estudadas por estudantes de programação, pois ela é de fácil aprendizado e fácil de visualizar a evolução.

Príncipais TAGS:

HEAD - Tudo que ficar dentro da tag head não irá aparecer no seu site, será apenas para Metadados, links, etc.

BODY - Aqui é onde se localiza o corpo do projeot. Tudo daqui para baixo fará parte da parte visual do projeto.

H1,H2,H3,H4,H5 - Tags utilizadas para títulos / subtítulos importância de relevância.

P - tag utilizada para Parágrafos.

SPAN - Utilizada para parágrafos de texto sem quebrar linha. Também utilizada para marcar determinada palavra em específico para que posas ser estilizada posteriormente no CSS.

BR - Quebra de Linha.

HR - Gera uma linha visível na página.

A - Tag que cria um LINK. Podendo ter o target normal ou target_blank. Literalmente um link para uma futura página, dentro ou fora do projeto.

NAV - Tag de Área de navegação. Utilizada para Menu's por exemplo.

IMG - Tag utilizada para inserir Imagem em uma página.

FIGURE - Serve para agrupar míodias (imagem, gráficos, etc)

FIGCAPTION - Legenda de Mídias.

VIDEO - Inserir Vídeos

AUDIO - Inserir Áudios no projeto.

DIV - Serve para gerar um Container, sem significado semântico. Containers servem para agrupar tags

HEADER - Container com signfiicado semantico para o cabeçalho da página

MAIN - Container com significado semantico para o conteúdi principal da página

ARTICLE - Conteúdo independente da página como um Post, Card, Notícia ( menos relevante)

ASIDE - Conteúdo lateral da página.

FOOTER - Conteúdo semantico para o Rodapé da página ou Seção

UL - Abre o espaço para uma lista não ordenada

OL - Abre o espaço para uma lista ordenada

LI - Cada linha entro das tags acima de uma lista.

FORM - Abre um container de Formulário

INPUT - Campo de entrada de texto ( texto, senha, email, etc)

LABEL - Rótulo do Campo.

TEXTAREA - Texto Longo.

SELECT - Lista Suspensa.

OPTION - Será cada campo dentro dessa lista suspensa.

BUTTON - Botão.

LINK - Tag para importar CSS, Fontes etc.

META - Tags de Metadados como charset, viewports, SEO'S , etc

STYLE - Tag utiizada para CSS Internos.

SCRYPT - Tag utilizada para importar Javascript dentro do Projeto HTML.

STRONG - Texto com enfase forte, semantica não só visual.

EM - Enfase leve semantica, italica.

MARK - Texto marcado

TIME - Datas e Horários.
