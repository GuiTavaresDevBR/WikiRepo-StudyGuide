JavaScript Básico

JavaScript é a linguagem de programação obrigatória para o frontend. Com o tempo, será possível perceber que se você tiver um bom pilar, as demais linguagens são bem parecidas. Todos os frameworks mais utilizados no mercado (React, Vue, Angular) são frameworks da linguagem JS. Um dos maiores erros da programação é começar estudando direto o framework sem ter uma boa noção da linguagem em si. Um profissional diferenteciado é o que sabe a linguagem a fundo. Aqui iremos abordar os principais tópicos do JS.

Tópicos JS Básicos:

Tipos de Dados: aqui veremos os timos de dados existentes em JS:

  • STRING - "texto"
  • NUMBER - "10, 3.5"
  • BOOLEAN - true / false
  • UNDEFINED - variável declarada sem valor
  • NULL - ausencia intencional de valor
  • BIGINT - números muito grandes

Objetos: Tudo o que não é primitivo dentro do JS é um objeto. Object, Arrays, Functions, Dates. Tudo isso é considerado um objeto. Porém dentro da programação é muito como nos depararmos com objetos do tipo:

  • let pessoa = { nome: "Guilherme", idade: 25 };

Variáveis: É a declaração na qual poderemos armazenar os dados. No JS elas não são fortemente tipadas. As mais utilizadas são:

  • LET - pode mudar valores
  • CONST - Valores não podem ser alterados.

Condicionais Aqui as condicionais funcionam basicamente igual nas demais linguagens a base de IF/ELSE.

  • if( idade > 18){console.log("deu certo")}else{...}

Funções funções em javascript podem ser declaradas basicamente de três formas:

  • TRADICIONAL: function soma(a,b){ return a + b}
  • ARROW FUNCTION: const soma = (a + b ) => a + b

Arrays Arrays em JS sao colocados entre [] EX: let lista = [1,2,3]. É de extrema importancia saber que os arrays possuem Métodos de fundamental importancia para saber manusea-los

  • push() - adiciona
  • pop() - remove o último
  • shift()- remove o primeiro
  • unshift() - adiciona no início
  • lenght - tamanho de posições do array.

Novidades ES6

Template Strings Forma de intercalar frase com variáveis através do símbolo de $.

  • let nome = "Gui"
  • console.log(`Olá ${nome}`)

Desestruturação Aqui é fundamental e muito utilziado nos dias de hoje. É uma forma de retirarmos dados de um objeto e já atribuirmos para variáveis específicas. por exemplo:

  • const pessoa = { nome: "Gui", idade: 20 };
  • const { nome, idade } = pessoa;

Dessa forma, teremos a variável nome e dade, com os respectivos valores retirados de dentro do objeto pessoa.

Métodos Arrays Modernos:

  • MAP - Transforma o Array - const dobro = [1,2].map(n => n*2)
  • FILTER - Filtra no Array - const maiores = [5,10].filter(n => n > 5)
  • REDUCE
  • FIND - Encontra - const item = array.find(x => x.id === 1)
  • FOREACH - Percorre o array - array.forEach( item => console.log(item))

Assíncrone em Javascript - JavaScript é Single-Thread, mas trabalha também com tarefas Assíncronas. Por exemplo, ao realizarmos uma requisicão para uma api externa, estaremos realizando uma tarefa Assíncrona, pois precisaremos esperar a resposta da API, que pode ser rapida ou demorar muito tempo para ser resolvida. Esse assunto é de extrema importância para um Dev, pois ao trabalhar no mercado real, isso será normal de uso no dia a dia.
