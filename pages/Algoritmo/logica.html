Lógica de Programação

Aqui iremos aprender os fundamentos de Programação. Os conceitos. Tendo os conceitos bem definidos, qualquer linguagem se torna mais fácil

Algorítimos: Sequencia lógica e finita de passos para resolver algum determinado problema.

Variáveis: Espaços na memória usados para armazenar valores que podem variar ou não durante a execução do programa. Variáveis possuem diferentes tipos, que variam de acordo com os tipos de dados, de cada linguagem de programação. Exemplos de uso:

  • Guardar idade
  • Guardar nome
  • Guardar resultado de um cálculo

Tipos de Dados São os tipos de dados que irão variar de linguagem para linguagem. Abaixo as principais utilizadas na maioria das linguagens:

  • Inteiro (INT) - Números Inteiros
  • Decimal (FLOAT/DOUBLE) - Números com casas decimais
  • Boolean (BOOLEAN / BOOL) - Verdadeiro ou falso
  • Caractere(CHAR) - Único síbolo
  • Texto / String (STRING) - Conjunto de caracteres

Constantes Valores que uma vez definitdos, não podem ser alterados durante a execução do programa.

Operadores Podem ser aritiméticos ou Relacionais.

  • + (Soma)
  • - (Subtração)
  • * (Multiplicação)
  • / (Divisão)
  • % (Resto da divisão)
  • == (Igual)
  • != (Diferente)
  • > (Maior)
  • >= ( Maior ou Igual)
  • && (E)
  • || (OU)
  • ! (NÃO)

In/Out Em programação é comum nos depararmos com Entrada de dados ( IN) , onde o usuário irá informar valores ao algoritmo como ler nome, ler idade, ler número etc. Também temos as saídas de dados, que é quando o programa exibe informações para o usuário.

Estruturas Condicionais Permite executar diferentes blocos de códigos de acordo com uma condição. Usado sempre quando: "Se isso acontecer, faça aquilo". É a responsável pelo fluxo do algoritmo.

  • IF/ ELSE
  • ELSE IF

Estruturas de Repetição Permite que um determinado trecho de código seja repetido diversas vezes.

  • FOR : Repitção com controle de início e fim
  • WHILE: Repete enquanto a condição for verdadeira.
  • DO While: Executa pelo menos uma vez antes de testar a condição.

Funções Blocos de códigos reutilizáveis que executam uma tarefa em específica. Elas nos permite reutilização de códigos, códigos mais limpos, e uma melhor organização. Eles podem ou não enviar algum retorno. Mais para frente iremos perceber que ao trabalahrmos com frameworks front-end tudo será baseado a em funções. Funções podem também receber parâmetros, que são dados enviados ao chamar a função.

Vetores/Arrays Estruturas qeu armazenam vários valores do mesmo tipo. Muito utilizados para salvar listas.

Matrizes Vetores de duas ou mais dimensões. ( Tabela, planilha, tabuleiro)
